# CityCare Hospital Power BI Dashboard
## Project Overview
This project presents a Hospital Analytics Dashboard developed using Microsoft Power BI.
The dashboard helps hospital management monitor patient admissions, discharge trends, billing performance, diagnosis distribution, doctor feedback, and bed occupancy.
---
## Dashboard Features
### KPIs
- Admit Date
- Discharge Date
- Follow-up Date
- Billing Amount
### Visualizations
- Bed Occupancy Analysis
- Doctor Feedback Volume
- Diagnosis Type Distribution
- Billing Amount vs Health Insurance Amount
- Patient Filters
- Date Range Slicer
---
## Tools Used
- Microsoft Power BI
- Microsoft Excel
- Data Analytics
- Data Visualization
---
## Dataset
The project uses hospital patient records containing:
- Patient ID
- Admission Details
- Doctor Information
- Diagnosis Type
- Insurance Amount
- Billing Amount
- Bed Category
